# Dotfiles

Configuraciones de terminal y herramientas de desarrollo.

## Instalación

```bash
git clone https://github.com/TU_USUARIO/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
python3 install.py
```

## Contenido

| Categoría | Descripción |
|-----------|-------------|
| shell/    | zsh, bash configs |
| git/      | gitconfig, gitignore |
| tmux/     | tmux.conf |
| starship/ | Prompt config |
| gh/       | GitHub CLI |
| atuin/    | Shell history |
| btop/     | System monitor |
| lsd/      | ls replacement |
| zed/      | Zed editor |
| claude/   | Claude Code config |

## Restaurar en otra máquina

```bash
python3 install.py
```
